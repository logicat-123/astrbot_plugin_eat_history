# 赤石插件

缓存转发消息，发送 `/我要赤石`随机查看

## 命令

| 命令      | 说明                   |
| --------- | ---------------------- |
| /我要赤石 | 获取缓存的消息记录     |
| /查看史书 | 获取缓存的消息记录信息 |

## 缓存机制

- 仅缓存转发的聊天记录，其他消息不缓存
- 通过在astrbot后台管理页面中配置插件的黑名单或者白名单
- 在黑名单模式下，插件缓存除了黑名单之外群聊中转发的聊天记录
- 在白名单模式下，插件缓存白名单中包含的群聊中转发的聊天记录

## 赤石机制

- 从缓存的聊天记录中随机获取一条消息发送
- 消息每次发送后，下次被发送的权重降低
